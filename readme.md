# Rust Camera Processing

A simple camera processing application written in Rust using OpenCV.

## Prerequisites

- Rust (latest stable version)
- OpenCV development libraries
- A working webcam

### Installing OpenCV

#### macOS

```bash
brew install opencv
```

#### Ubuntu/Debian

```bash
sudo apt-get install libopencv-dev
```

## Building and Running

1. Clone the repository
2. Build the project:

```bash
cargo build
```

3. Run the application

```bash
cargo run
```

## Usage

- The application will open your default camera and display the video feed
- Press 'q' to quit the application

## Features

- Real-time camera capture
- Basic video display
- Simple keyboard controls
